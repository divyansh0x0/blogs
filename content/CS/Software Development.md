## Context of Software Development

It is environment, purpose and condition in which software is planned, used and maintained.

Main aspects of software development:

1. Problem Identification
2. Users and Requirements 
3. Hardware and software environment
4. Programming language and Tools
5. Development process
   - Analysis
   - Design
   - Coding
   - Testing
   - Maintainance
6. Constraint
