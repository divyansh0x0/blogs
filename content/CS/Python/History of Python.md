# Python

Python was conceived in the late **1980s** by Guido van Rossum at Centrum Wiskunde & Informatica (CWI) in the Netherlands.

It was a successor to ABC programming language.

The name Python derives from the British comedy series **Monty Python's Flying Circus**.

## Python releases

1. `Python 0.9.0`: Python implementation began in December 1989. It was first released in 1991 as python `0.9.0`.
2. `Python 2.0`: Released in 16 October 2000. This release provided new features such as:
    - List comprehensions
    - Cycle detecting
    - Reference counting
    - Unicode support
3. `Python 3.0`: It was released in 3 December 2008, and was a major revision and not completely backward compatible with earlier verisions.

> The support for python 2.0 was dropped after 2.7 whose end of life was initially set for 2015 but was postponed to 2020
